API Endpoints

Endpoint Reference

Key endpoints for operation. All protected endpoints require a Bearer Token.

Authentication

POST /api/auth/login
Description: Authenticates user credentials.

Body: { "email": "...", "password": "..." }

Returns: { "token": "jwt...", "expiration": "..." }

Tournaments

POST /api/tournaments
Description: Create new tournament. (Admin Only)

Body: { "name": "Cup", "joinDeadline": "iso-date", "xpPerWin": 100 }
GET /api/tournaments
Description: List all tournaments. (Admin Only)
POST /api/tournaments/{id}/join
Description: Public registration. Rate limited to 10 requests/min.

Body: { "walkerId": 123, "walkerName": "Name" }

Matches

POST /api/matches/{id}/winner
Description: Set match winner. Triggers auto-advancement logic. (Admin Only)

Body: { "winnerId": "guid" }
